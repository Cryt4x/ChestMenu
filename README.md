# ChestMenu
A plugin which provides (soon a customizable) Chest Menu f.e. game-modes etc.
CUI (Chest User Interface)

# Currently Options
1. Changing Game-Mode to:
- Survival
- Creative
- Adventure
- Spectator
2. Switching to next Menu (WIP)
3. Opening Menu with "/ti" (ti =: technical interfce)
